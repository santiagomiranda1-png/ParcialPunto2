# Proyecto Punto2

Resumen
-------
Este repositorio contiene una pequeña aplicación en Python. El archivo principal es `main.py`.

Requisitos
---------
- Python 3.8+
- Dependencias listadas en `requeriments.txt`

Instalación
-----------
1. Crear y activar un entorno virtual (recomendado):

```bash
python3 -m venv venv
source venv/bin/activate
```

2. Instalar dependencias:

```bash
pip install -r requeriments.txt
```

Ejecución
--------
Para ejecutar la aplicación:

```bash
python main.py
```

Estructura del repositorio
--------------------------
- `main.py` — Punto de entrada de la aplicación.
- `requeriments.txt` — Dependencias Python.
- `__pycache__/` — Caché de Python (ignorada normalmente).
